# Installation

The Binboi CLI is currently distributed as source. Prebuilt packages for Homebrew, a curl installer, and Windows are on the roadmap.

## Requirements

- Git
- Go (matching the version in the repo's `go.mod`)
- `make`

## Build from source

```bash
git clone https://github.com/Miransas/binboi.git
cd binboi
make build-cli
sudo mv dist/binboi /usr/local/bin/
```

## Verify

```bash
binboi --version
```

If the command is not found, confirm that `/usr/local/bin` is on your `PATH`.

## Upgrade

Pull the latest source and rebuild:

```bash
cd binboi
git pull
make build-cli
sudo mv dist/binboi /usr/local/bin/
```
