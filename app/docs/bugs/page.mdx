# Report a Bug

We track bugs on GitHub.

## Before Filing

- Search [existing issues](https://github.com/Miransas/binboi/issues) — yours may already be reported
- Try the [troubleshooting guide](/docs/troubleshooting)
- Upgrade the CLI: `binboi update`

## File an Issue

[github.com/Miransas/binboi/issues/new](https://github.com/Miransas/binboi/issues/new)

Include:

- CLI version (`binboi --version`)
- Operating system (`uname -a` or `ver`)
- The exact command you ran
- The full output, including error messages
- What you expected to happen

For control-plane issues, include the `x-request-id` header from the response — it's how we trace requests internally.

## Security Issues

Do not file public issues for security bugs. Email [security@miransas.com](mailto:security@miransas.com) with details. We aim to respond within 48 hours.

## Feature Requests

Use [GitHub Discussions](https://github.com/Miransas/binboi/discussions) for ideas and proposals.
