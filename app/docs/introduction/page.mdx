# Introduction

Binboi is a tunneling service that exposes a local port to the public internet over HTTPS. You run a binary on your machine, Binboi gives you a `https://<subdomain>.binboi.com` URL, and traffic to that URL is forwarded to your local process.

## What you can do with it

- Share a local web app with teammates or clients before you deploy it.
- Receive webhooks from Stripe, GitHub, Slack, or any other provider on your laptop.
- Test mobile apps against a local backend without changing networks.
- Demo work-in-progress without standing up a server.

## Key features

- HTTPS by default on every tunnel — TLS terminates at the Binboi edge.
- Authentication via Personal Access Tokens (PATs) — no passwords in the CLI.
- Tunnel state and traffic visible at [binboi.com/dashboard](https://binboi.com/dashboard).
- REST API at [api.binboi.com](https://api.binboi.com) for programmatic control.
