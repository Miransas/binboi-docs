# Quick Start

Get a local port exposed at `https://<subdomain>.binboi.com` in four steps.

## 1. Create an account

Sign up at [binboi.com](https://binboi.com) and verify your email. New accounts start on the Free plan.

## 2. Generate a Personal Access Token

Open [binboi.com/dashboard/access-tokens](https://binboi.com/dashboard/access-tokens) and click **Create token**. The token is shown once in a modal with a **Copy** button — copy it now and store it somewhere safe. It cannot be retrieved later.

Token format:

```
binboi_pat_<8 hex>_<32 hex>
```

The Free plan allows one PAT for the lifetime of the account.

## 3. Install the CLI

The CLI is currently built from source. Homebrew and a curl installer are coming.

```bash
git clone https://github.com/Miransas/binboi.git
cd binboi
make build-cli
sudo mv dist/binboi /usr/local/bin/
```

Verify:

```bash
binboi --version
```

## 4. Log in and open a tunnel

```bash
binboi login --token binboi_pat_xxxxxxxx_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
binboi http 3000 my-app
```

Your local service on port `3000` is now reachable at:

```
https://my-app.binboi.com
```

Press `Ctrl+C` to stop the tunnel.
